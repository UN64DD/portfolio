## Features

- **Hero Section** - Introduction and quick links to other sections
- **Skills** - Showcase of technical skills and technologies
- **Testimony** - Recommendations and testimonials
- **Contact** - Contact form for reaching out

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome 6.4.0
- Google Fonts (Poppins)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/UN64DD/portfolio.git
   ```

2. Open `index.html` in your browser, or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

3. Visit `http://localhost:8000` in your browser.

## Customization

- **Replace images** - Update the files in the `images/` folder
- **Edit content** - Modify the text in `index.html`
- **Change styles** - Edit `styles.css` to customize colors, fonts, and layout
